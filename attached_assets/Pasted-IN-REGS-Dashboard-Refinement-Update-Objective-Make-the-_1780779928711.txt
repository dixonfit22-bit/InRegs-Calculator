IN REGS Dashboard Refinement Update

Objective:
Make the dashboard instantly understandable to Marines and leadership while ensuring readiness statistics are logically accurate.

PRIMARY GOAL:
A user should be able to open the dashboard and understand unit readiness within 5 seconds.

⸻

1. FIX CATEGORY LOGIC

⸻

Current issue:

In Regs = 0
Watch Zone = 1
Out of Regs = 0

This is misleading because Watch Zone Marines are still passing and still within standards.

Update logic:

Passing/In Regs = ALL Marines currently meeting standards.

Watch Zone = subset of Passing/In Regs.

Example:

1 Marine at 19.3% with a 20% limit:

Pass Rate = 100%
Passing = 1
Watch Zone = 1
Out of Regs = 0

Never display a scenario where:
Pass Rate = 100%
In Regs = 0

because that is logically incorrect.

⸻

2. SIMPLIFY COMMAND DASHBOARD

⸻

Replace unnecessary text with quick-read metrics.

Keep:

* Total Marines
* Pass Rate
* Passing
* Watch Zone
* Out of Regs

Remove:

* Excessive explanatory text
* Long readiness descriptions
* Unnecessary dashboard clutter

Dashboard should feel like a readiness snapshot, not a report.

⸻
